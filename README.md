# mehrawbmh.github.io

Personal portfolio site for **Mehrab Mohammad-Hosseini** — Software Engineer.

Live at: [mehrawbmh.github.io](https://mehrawbmh.github.io)

## Tech

Vanilla HTML5, CSS3, and JavaScript — no frameworks, no build step, no dependencies. Hosted on GitHub Pages.

## Run locally

```sh
python3 -m http.server 8000
```

## Structure

```
├── index.html        # All page content
├── css/style.css     # Styles & CSS variables
├── js/main.js        # Scroll effects, nav, animations
├── assets/           # Images
```
